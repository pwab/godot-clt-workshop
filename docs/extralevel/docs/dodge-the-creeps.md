!!! info "Wie geht das?"
    Um ein Spiel in eine Webseite einzubauen, braucht es verschiedene Dinge:
    
    1. Ein Spiel (bspw. die Demo von Dodge the Creeps)
    2. In Godot installierte Export Templates (gibt es [hier](https://godotengine.org/download))
    3. Ein HTML5 Export des Spiels (muss man [einrichten](../../topics/export))
    4. Ein Hoster für den Upload der Dateien (in diesem Fall [GitHub](https://github.com/pwab/dodgethecreeps))
    5. Einen Webseitenprovider der die Dateien bereitstellt (in diesem Fall [GitHub Pages](https://pwab.github.io/dodgethecreeps/))
    6. Eine eigene Webseite um den Link zum Spiel als iframe einzubetten (diese Workshopseite hier)

    Dann kann man die meisten Spiele problemlos für den Webbrowser exportieren und irgendwo in eine Seite einbetten. Viele Leute nutzen dafür [itch.io](https://itch.io/), weil es eine einfache Möglichkeit bietet, sein Spiel anderen Menschen zu zeigen, Feedback zu bekommen oder sogar etwas Geld zu verdienen. Schau dich dort mal einfach um und gib beispielsweise _Godot_ in die Suchleiste ein. Da gibt es viel zu sehen!
